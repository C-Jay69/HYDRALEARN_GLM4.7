# **App Name**: Lifejacket AI

## Core Features:

- Personalized Lesson Generation: Generates adaptive lesson plans for each student based on diagnosed needs and pulls from official curriculum, ensuring class cohesion.
- Automatic Assessment Engine: Create an automatic assessment tool, to deliver quizzes, games, projects, and essays for competency tracking.
- Teacher Dashboard: Provides a dashboard that offers granular performance data and suggests micro-plans for students needing extra help.
- Gamified Learning: Facilitates in-class competitive games with real-time student vs. AI or group vs. group activities, rewarding progress with badges and updates.
- Teacher Assistant Mode: Automates lesson scheduling, exam creation, and marking with AI, while also suggesting differentiated activities for students.
- Equity and Inclusion Engine: Supports ESL/ELL students and dyslexia-friendly content with translation mode for bilingual lesson plans.

## Style Guidelines:

- Primary color: HSL 200, 60%, 50% (RGB hex: #3399CC) evokes reliability and a sense of calm, mirroring the app's aim to alleviate stress. 
- Background color: HSL 200, 20%, 95% (RGB hex: #F0F8FA) provides a soft, unobtrusive backdrop for focused learning. 
- Accent color: HSL 170, 50%, 40% (RGB hex: #33A699) contrasts effectively, highlighting interactive elements and key information without overwhelming the user. 
- Body text: 'PT Sans', a humanist sans-serif for a modern and accessible reading experience
- Headlines: 'Space Grotesk', a geometric sans-serif, for impactful headings. 
- Use clean, modern icons that visually represent different subjects and functionalities.
- Employ subtle transitions and feedback animations to enhance user interaction without distraction.
